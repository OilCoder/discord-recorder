# Discord Recorder — Vampiro

Bot de Discord que graba el audio de voz durante las partidas, lo transcribe con Whisper y genera un resumen de los hechos para el master.

## Contenido

- `bot/` — Código del bot (grabación de voz)
- `plan/` — Diseño técnico: bot, transcripción y resumen LLM
- `output/` — Archivos de audio generados
- `main.py` — Punto de entrada
- `docker-compose.yml` / `Dockerfile` — Despliegue
